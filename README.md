# 📈 Student Performance Predictor — Linear Regression

Проект по машинному обучению: предсказание успеваемости студентов
с помощью множественной линейной регрессии.
Включает обучение модели, предсказание через терминал и веб-сайт на Flask.

---

## 📁 Структура проекта
```
linear_regression_project/
│
├── train_model.py       ← Шаг 1: обучение и сохранение модели

```

## 🗃️ Датасет

**10 000 строк**, 5 признаков + 1 целевая переменная.

| Столбец | Описание | Диапазон |
|---|---|---|
| `Hours Studied` | Часов учёбы в день | 1 – 9 |
| `Previous Scores` | Предыдущие оценки | 40 – 99 |
| `Extracurricular Activities` | Внеклассные занятия | Yes / No |
| `Sleep Hours` | Часов сна | 1 – 9 |
| `Sample Question Papers Practiced` | Пробных тестов решено | 0 – 9 |
| `Performance Index` | **Итоговый балл** — целевая переменная | 0 – 100 |

---

## 🧠 Модель

- **Алгоритм:** Multiple Linear Regression (scikit-learn)
- **Признаки (X):** 5 штук (см. таблицу выше)
- **Цель (Y):** `Performance Index` (итоговый балл студента)
- **R² Score:** ~0.989 — модель объясняет 98.9% дисперсии данных
- **RMSE:** ~2.02 балла — средняя ошибка предсказания
- **Разбивка:** 80% обучение / 20% тест

### Формула модели:
```
Performance Index =
    2.85 × Hours Studied
  + 1.02 × Previous Scores
  + 0.61 × Extracurricular Activities
  + 0.48 × Sleep Hours
  + 0.19 × Sample Question Papers Practiced
  - 33.92
```

## 🚀 Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/TarielevAlihan/student-performance-predictor.git
cd student-performance-predictor
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Обучить модель
```bash
python train_model.py
```
Создаст файл `model.pkl` с обученной моделью и выведет метрики.

### 4. Предсказание через терминал
```bash
python predict.py
```

### 5. Запустить веб-сайт
```bash
python app.py
```
Открой браузер: **http://127.0.0.1:5000**

## 💻 Использование

### Терминал (`predict.py`)

Программа задаёт вопросы и выводит предсказание:
```
Загружаем модель...
Модель загружена!
   R²: 0.9890  |  RMSE: 2.02

Введите данные студента:
   Часы учёбы в день (1–9): 7
   Предыдущие оценки (40–99): 85
   Внеклассные занятия? (yes/no): yes
   Часов сна (1–9): 8
   Решено пробных тестов (0–9): 4

────────────────────────────────────────
Результат предсказания:
Performance Index: 88.3 / 100
Оценка:            B
────────────────────────────────────────
```

### Веб-сайт (`app.py`)

- Слайдеры для ввода всех 5 признаков
- Кнопки **Yes / No** для внеклассных занятий
- Прогноз балла с оценкой **(A / B / C / D / F)**
- Графики регрессии по каждому признаку
- Визуальный индикатор точности **R²** от 0 до 1
- Таблица датасета (первые 200 строк)
