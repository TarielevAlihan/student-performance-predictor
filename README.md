# 📈 Linear Regression — Прогноз оценок учеников по многим критериям

Flask-сайт с моделью линейной регрессии (scikit-learn).

---

## 📁 Структура проекта

```
linear_regression_project/
│
├── train_model.py      ← шаг 1: обучение и сохранение модели
├── app.py              ← шаг 2: Flask-сервер + предсказания
├── model.pkl           ← сохранённая модель (создаётся после шага 1)
├── dataset.csv         ← датасет (создаётся после шага 1)
├── requirements.txt    ← зависимости
│
└── templates/
    └── index.html      ← веб-интерфейс
```

---

## 🚀 Запуск

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Обучить и сохранить модель
```bash
python train_model.py
```

### 3. Запустить сайт
```bash
python app.py
```

Открой браузер: **http://127.0.0.1:5000**

---

## 🔍 Что делает проект

| Файл | Назначение |
|------|-----------|
| `train_model.py` | Генерирует датасет, обучает LinearRegression, сохраняет `model.pkl` |
| `app.py` | Загружает модель, строит граф, принимает POST-запросы с предсказанием |
| `index.html` | Интерфейс: ввод информации → прогноз оценки + обновляемый график |

---

## 📊 Модель

- **Алгоритм:** Linear Regression (scikit-learn)
- **Признак1 (X):** Часов учёбы в день (1–9)
- **Признак2 (X):** Предыдущие оценки и тд (40–99)
- **Признак3 (X):** Внеклассные занятия(y or n)
- **Признак3 (X):** Часов сна (1–9)
- **Признак3 (X):** Пробных тестов решено (0–9)
- **Цель (y):** оценка ученика 
- **R²:** ~0.986
